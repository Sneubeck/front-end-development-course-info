List Style

Changes the default styling of UL and OL lists

  • list-style-type - changes the icon/value used in the list
  • list-style-position - changes the position of the list. Accepts inside and outside
  • list-style-img - changes bullets to image. Accepts none or a valid URL
  • Short handlist-style: type position img

Grouping Selectors

You can leverage grouping selectors to share styles between multiple selectors. This can be done by adding a comma between each selector. 


    h1, h2, .header, #site-header{
        color:#ff0000;
    }

What are psuedo-classes?

Listing a directory

ls


$ ls

MyForks/   presentations/  'year 1 slides'/

Changing a Directory

cd


#list directory
$ ls

MyForks/   presentations/  'year 1 slides'/
#change directory into MyForks
$ cd MyForks/
$ ls

project1/ project2/

Directory Shortcuts

  • . - Current Directory
  • .. - Parent Directory
  • ~ - User Home - C:\Users\jisajmb
  • / - Root

Displaying your current path.

Got lost in your console? Use the print working directory command to get a full path of your current location

pwd


$ cd ~
$ pwd
c/users/jisajmb

Creating a directory/folder

You can create a new folder by using the make directory command.

mkdir


#list directory
$ ls

MyForks/   presentations/  'year 1 slides'/

$ mkdir my-work 
$ ls

my-work/  MyForks/   presentations/  'year 1 slides'/

Make a file via command line

touch


#list directory
$ ls

my-work/ MyForks/   presentations/  'year 1 slides'/

$ cd my-work 
$ ls


$ touch index.html
$ ls

index.html

The Clear Command

You can clear the console by simply typing clear and hitting enter.
